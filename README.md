# SwiftUI: In App Purchase Demo

A demo of supporting in app purchase for both products as well as subscriptions. 
Including 
- processing transactions
- tracking updates
- Getting current purchased product/subscriptions
- handling refunds

To set up testing in Xcode, select scheme, choose Edit Scheme, and under Run > Options, choose the configuration file to test on under StoreKit Configuration.
